This directory contains train-validation datasets created from the Bangla-Lekha-Isolated dataset. The dataset was split into train and test set in 80:20 ratio and the train set was further split into 80:20 - train:validation sets. All split was done keeping the 'good', 'bad' label ratio stratified.

'aesthetics_train.csv' contains aesthetic_quality label of each train sample

'aesthetics_val.csv' contains aesthetic_quality label of each validation sample

'aesthetics_test.csv' contains aesthetic_quality label of each test sample

Folders- 'train_images', 'validation_images' and 'test_images' contains sample images for the training, validation and test set respectively.
